4. CORE SCORE FORMULA (Sacred- Immutable)
4.1 Core Score Calculation:
RAW = (APS * 0.28) + (DRI * 0.24) + (RVS * 0.20) + (URI * 0.15) + (TSI * 0.13)
PQS Penalty:
IF PQS >= 700: penalty = PQS * 0.20
IF PQS >= 400: penalty = PQS * 0.10
IF PQS < 400: penalty = 0
BACKEND = ROUND(MAX(0, MIN(1000, RAW- penalty)))
DISPLAY = ROUND(BACKEND / 10.0, 1) // 0.0 to 100.0
4.2 Patient Classification (from DISPLAY score):

4.3 60/40 LTV Weighted Rule:
IF months_absent > 18:
MODE = 'first_time'
SCORE = session_score
ELSE:
MODE = 'weighted_ltv'
SCORE = ROUND(historical_avg * 0.60 + session_score * 0.40)
4.4 DISC Profile:
• Stored in patient_longitudinal_profiles.dominant_disc_profile.
• Values: driver, influencer, analytical, emotional, or NULL.
• Derived from survey responses + session behavior- NOT manual entry.
PROTECTION:
• These weights are IMMUTABLE. No AI may suggest changing them.
• The formula MUST be implemented in core/rules/scoring/CoreScoreEngine.ts.
• Edge Function score-calculator MUST use this exact formula.
• ANYdeviation from these weights breaks the behavioral analysis.
